Simulacion Ctrl IoT

Breve descripción:

Esta aplicación fue elaborada con la intención de realizar una representación de un control con IoT, permite manipular un cierto rango de valores y asignarlos a la interfaz principal, mostrando mediante un input range, el porcentaje el cual fue manipulado en la página de simulación.
Los cambios de los valores son reflejados en un historial que se muestra dentro de la aplicación y para agregar, esta misma es una App Web Progresiva (PWA).

Tecnológias utilzadas:

Javascript

HTML

CSS

Firebase

Material Design
